# 认知交付与课程逐字稿

## 单章五要素

1. 认知起点：学员已有的理解、经验或常见误判。
2. 认知终点：本章结束时建立的新判断。
3. 挑战情境：具体场景、问题与约束。
4. 感知素材：有来源的案例、故事、数据与讲师原话。
5. 收获和行动：学员能带走的产物、动作或判断标准。

单章还必须说明真实教学任务：建立全景、纠正误区、给方法模板、训练判断，或衔接前后章节。宽泛“讲清某概念”通常不够。

## 连续讲课结构

开头同时承接前课、说明本课任务、提出贯穿判断。中段让案例、方法和边界沿一条解释链推进；强判断在开头出现，中段被证据解释，结尾再回收。本章产物要能进入下一章。

一个自然段承载一个教学动作，通常由 2–5 个短句组成句群。强调、转场、课堂提问、例句、作业和金句回收可以单独成段。连续解释不要写成一行一句的字幕。

## 证据边界

事实、数字、案例、背书与逐字原话必须指向知识卡。缺证据时保留待确认占位。讲师观点可明确署名，不要改写为无主语的普遍真理。
