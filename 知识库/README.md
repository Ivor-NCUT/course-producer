# Course Producer 知识库

这里保存课程生产阶段共同使用的方法、模板、概念词典和原创知识原子。主路由不一次性加载全部文件；每个原子 Skill 只读取当前阶段明确引用的资源。

## 目录

- `方法论/`：定位、访谈、知识萃取、成果优先课纲、逐字稿、审校、飞书协作与海报。
- `模板/`：课程项目、访谈记录、知识卡、单课、审校记录和海报方案。
- `原子库/atoms.jsonl`：本项目原创、带来源和置信度的最小知识单元。
- `课程制作词典.md`：跨 Skill 统一术语与状态口径。

客户课程材料和项目级审美偏好不进入这里，默认只保存在用户项目的 `.course-producer/`。外部事实、案例和数据必须保留原始来源；无法确认再分发权的内容只记录引用，不复制全文。

## JSONL 规则

`atoms.jsonl` 一行一个 JSON 对象，至少包含 `id`、`knowledge`、`type`、`topics`、`source` 和 `confidence`。新增原子前先检查是否已经存在同义项；来源不清或只是一次性项目经验时不要加入。
