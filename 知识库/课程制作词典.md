# 课程制作词典

| 术语 | 定义 |
|---|---|
| 课程定位 | 课程主题、目标学员、挑战、学习成果、使用场景与边界的可验证约定。 |
| 挑战情境 | 具体场景、当前问题和现实约束的组合。 |
| 认知起点 | 学员进入一章前已有的经验、俗知俗见或错误判断。 |
| 认知终点 | 学完一章后应建立的新判断。 |
| 感知素材 | 支撑教学判断的案例、故事、数据和讲师原话。 |
| 独立成果 | 学员只完成一个模块也能拿走并使用的产物、动作或判断。 |
| 螺旋复用 | 后章复用前章概念或产物，并在新约束下升级。 |
| 知识卡 | 带来源和证据位置的最小完整课程知识单元。 |
| source_id | 一项原始材料在 `materials.jsonl` 中的稳定标识。 |
| evidence_locator | 能回到原始证据的页码、时间戳、标题路径或 block id。 |
| 待验证主张 | 具有事实形式，但当前缺少充分证据的说法。 |
| 章节检查点 | 单章输入、产物、指纹和验证结果的恢复单元。 |
| 作品满意 | 用户确认当前版本可用，不等于长期规则成立。 |
| validated 偏好 | 经过多样本证据、反例和回测门后允许长期使用的审美规则。 |
| 完成度审计 | 对阶段状态、真实产物、证据链和必需交付逐项复核。 |
